# Portafolio Personal - Izan (Nomix7)

Bienvenido al repositorio de mi página web personal. Este proyecto es un portafolio diseñado con estilo Bento Grid (inspirado en la UI moderna y Discord), totalmente responsivo y optimizado.

**Ver Demo en vivo:** [https://nomix7.github.io/portfolio/](https://nomix7.github.io/portfolio/)

## Tecnologías Usadas
* **HTML5:** Estructura semántica.
* **CSS3:** Variables CSS, Grid Layout (Bento) y Flexbox.
* **FontAwesome:** Iconos vectoriales.
* **Google Fonts:** Tipografía 'Outfit'.

## Características
* **Diseño Bento Grid:** Organización modular del contenido.
* **Discord Theme:** Colores inspirados en Discord.
* **Responsive:** Se adapta a móviles y a ordenadores.
* **Integraciones:**
    * Reproductor de Spotify.
    * Mi Bot de Discord.
    * Enlaces directos a redes sociales.
    * Formulario de contacto directo a Gmail.
